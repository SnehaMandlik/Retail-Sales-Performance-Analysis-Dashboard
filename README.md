# Retail Sales Performance Analysis Dashboard

## Project Overview
This project analyzes BlinkIT sales data using Microsoft Excel to identify sales trends, outlet performance, product category performance, and customer purchasing patterns.

## Objective
- Analyze overall sales performance
- Identify top-selling product categories
- Compare outlet performance
- Generate business insights for decision-making

## Tool Used
- Microsoft Excel

## Excel Features Used
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Lookup Functions
- Data Cleaning
- Dashboard Design

## Key Metrics
- Total Sales
- Average Sales
- Number of Items Sold
- Average Rating

## Dashboard Insights
- Sales by Product Category
- Sales by Outlet Type
- Sales by Outlet Location
- Item Performance Analysis
- Customer Rating Analysis

## Key Findings
- Identified top-performing product categories.
- Compared sales across different outlet types.
- Analyzed customer ratings and purchasing trends.
- Generated insights to support business decisions.

- Excel Dashboard Creation
- Data Cleaning
- Data Analysis
- Data Visualization
- Business Reporting
